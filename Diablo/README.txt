Simply fetches the server status for Diabli 3 from http://eu.battle.net/d3/en/status
Only outputs status of Game Server, Gold (AH) and Hardcore (AH)

Example:
11:58:38 <@Hoaas> !diablo
11:58:39 <@Bunisher> Diablo III server (Europe): ▼ Maintenance. AH gold / hardcore: ▼ Maintenance / ▼ Maintenance.

11:59:31 <@Hoaas> !diablo am
11:59:33 <@Bunisher> Diablo III server (Americas): ▲ Available. AH gold / hardcore: ▲ Available / ▲ Available.
