Look up Norwegian municipalities (kommuner) by name or number. Data is fetched
live from [Norges kommuner on Norwegian Wikipedia](https://no.wikipedia.org/wiki/Norges_kommuner)
and cached for 24 hours.

## Commands

- `kommune <name>` — look up by name (exact, prefix, or substring match)
- `kommune <number>` — look up by 4-digit municipality number
- `kommune random` — show a random municipality

## Example output

```
0301 - Oslo (Adm. senter Oslo) i Oslo. 724 290 innbyggere. 454,03 km². Målform: nøytral.
```
